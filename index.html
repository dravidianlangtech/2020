Overview

The broader objective of DravidianLangTech-2021 will be 

  • To investigate challenges related to speech and language resource creation for Dravidian languages.

  • To promote a research in speech and language technology in Dravidian languages. 

  • To adopt an appropriate language technology models which suit Dravidian languages

  • To provide opportunities for researchers from the Dravidian language community from around the world to collaborate with other researchers.

We also organize shared task along with this workshop

  • Sentiment Analysis in Dravidian Code-Mixed Text- Tamil, Malayalam, and Kannada (Previous shared task link: https://dravidian-codemix.github.io/2020/). New to this task: Kannada language data and more sentences. Estimated Number of Participants: 120 (based on first shared task 2020)

  • Offensive Language Identification in Dravidian Code-Mixed Text- Tamil, Malayalam, and Kannada (Previous shared task link: https://sites.google.com/view/dravidian-codemix-fire2020/overview ).  New to this task: Kannada language data and more sentences. Estimated Number of Participants: 90  (based on first shared task 2020)

  • Meme classification for Tamil and Malayalam. Memes in Tamil and Malayalam will be given along with text. Participants have to predict whether a meme is troll or not based on multimodal content [3]. Estimated Number of Participants: 50

These shared tasks target the south Dravidian language on social media content, in future these shared tasks will extend to other Dravidian languages.
